# Simple Golang Web APP
## Propose
- Learn
- Practice

## Main Features
- Clean code with clean architecture
- SOLID
- Isolated repository
- Postgres DB
- Simple CRUD of Products with soft delete
- DDD
- Strategy

## How to use
- Create `.env` from `.env.example` (cp .env.example .env) 
- Change `.env` to use your postgres configuration
- run `go run main.go`
- Open `localhost:8000`
